# Práctica: Maquetación Web Thermicondado

###  Descripción
Página web temática sobre ventanas de PVC diseñada para practicar estructuras modernas de CSS.

###  Tecnologías Aplicadas
* **HTML Semántico:** Uso de etiquetas estructurales para SEO y accesibilidad.
* **CSS Grid:** Utilizado para el layout principal (Header, Main, Footer).
* **Flexbox:** Aplicado en el menú de navegación, las tarjetas de ventajas y el formulario de contacto.
* **Diseño Responsive:** Implementación de Media Queries para dispositivos móviles.
